Java Swing Calculator

A simple GUI-based Calculator application built using Java Swing. This
project performs basic arithmetic operations such as addition,
subtraction, multiplication, and division.

Features

-   Clean and simple user interface
-   Built using Java Swing components
-   Supports: Addition, Subtraction, Multiplication, Division, Decimal
    numbers
-   Clear (C) button
-   Interactive button-based input
-   Dynamic display label to show results and numbers

Technologies Used

-   Java
-   Java Swing (JFrame, JButton, JLabel)
-   Event Handling (ActionListener)

Project Structure

calculator/ └── Calc.java

How to Run

Using Eclipse

1.  Create a new Java project
2.  Create a package named calculator
3.  Add Calc.java inside the package
4.  Run Calc.java → Run As → Java Application

Using Terminal

1.  Navigate to the project folder: cd src
2.  Compile: javac calculator/Calc.java
3.  Run: java calculator.Calc
